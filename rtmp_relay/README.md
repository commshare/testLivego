# rtmp_relay
rtmp relay server：
==================================================
* Platform：linux

* How to use?<br/>
example:
xxx# ./rtmp_relay rtmp://x.x.x.x/live/source rtmp://x.x.x.x/live/target<br/>
first parameter: source rtmp server url<br/>
second parameter: destination rtmp server url

dependance: boostlib
